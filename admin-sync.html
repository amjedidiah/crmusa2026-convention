How to Export from Zeffy

  1. Log into Zeffy → Fundraising → Payments
  2. Filter by your convention donation form and set your date range
  3. Click Export → Itemized Payments
  4. Make sure to include answers to custom questions — this is where the pledge code lives
  5. Drop the downloaded CSV file below
