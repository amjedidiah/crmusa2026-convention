Step 2 — Export from Zeffy

How to export from Zeffy:
  1. Log into Zeffy → Fundraising → Payments
  2. Filter by your convention donation form
  3. Set date range (e.g. last 7 days or all time)
  4. Click Export → choose Itemized Payments
  5. Make sure to include answers to custom questions — this is where the pledge code lives
  6. Download and drop the CSV below
